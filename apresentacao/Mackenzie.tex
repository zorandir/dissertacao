\section{Introdução}

\begin{frame}
    \frametitle{Introdução}
    \framesubtitle{}

    \texttt{Beamer} é uma classe de documentos do \LaTeX\ que gera apresentações para serem utilizadas com um projetor ou serem impressas em transparências.
    
    Criar uma apresentação utilizando a classe \texttt{Beamer} se parece muito com a criar um documento em \LaTeX\, existem as seções, as subseções e as páginas (slides) são chamados de frames.
    
    Você pode utilizar todo seu conhecimento de \LaTeX\ para criar apresentações utilizando o \texttt{Beamer}, sem a necessidade de reaprender a utilizar uma nova ferramenta.
    
    Caso você não conheça o \LaTeX\ é uma ótima oportunidade para conhece-lo. \cite{Tantau2013}
\end{frame}

\section{Tema do Mackenzie}

\begin{frame}
    \frametitle{Tema do Mackenzie}
    \framesubtitle{}
    Este tema foi desenvolvido tendo como base o tema da Universidade de Nebraska--Lincoln (UNL)\cite{Bourke2013}, que foi baseado no
    tema padrão chamado PaloAlto da classe \texttt{Beamer}

    Na versão da UNL foram acrescentados as seguintes características:
    \begin{itemize}
      \item A utilização das cores scarlet e creme.
      \item Melhorado o espaçamento.
      \item O modo matemático preserva as fontes serif do \LaTeX's.
      \item Adiciona automaticamente o logotipo da UNL.
      \item Sombras 
      \item Estilos de sombras projetadas nas barras superiores e laterais.
    \end{itemize}

\end{frame}
\begin{frame}
    \frametitle{Tema do Mackenzie}
    \framesubtitle{}
    No tema criado para o Mackenzie foram efetuadas as seguintes modificações:
    \begin{itemize}
      \item Alterado o padrão de cores Scarlet para o tom de vermelho do Mackenzie.
      \item Alterado o padrão de cores Creme para um tom mais claro melhorando o contraste com o vermelho.
      \item Adicionado um novo padrão de cor para as caixas de texto.
      \item Adiciona automaticamente o logotipo do Mackenzie.
    \end{itemize}

\end{frame}
    
\begin{frame}  %---------------------------------------------------------------
    \frametitle{Outras Características do Beamer}
    \framesubtitle{}

    As caixas de texto do Beamer por padrão já são arredondadas e tem uma sombra abaixo delas.  Todas as caixas de texto(definition, theorem, etc) Tiveram seu padrão de cores alterados para o novo estilo de cores do Mackenzie.

\section{Utilização do Tema}

\begin{frame}[fragile]
    \frametitle{Como posso utilizar o tema do Mackenzie?}
    \framesubtitle{Como passar opções para o tema?}

   Para utilizar o tema do Mackenzie insira a seguinte linha após a declaração de classe do seu documento:

    \begin{verbatim}\usetheme{Mackenzie}\end{verbatim}

    Para passar parâmetros para o tema, insira:

    \verb"\usetheme[left,hideothersubsections,width=2.5cm]{Mackenzie}"

\end{frame}

\begin{frame}[fragile]
    \frametitle{Como posso utilizar o tema do Mackenzie?}
    \framesubtitle{Como passar opções para o tema?}

    Existem vários parâmetros que podem ser passados ao tema:
    \begin{itemize}
      \item \texttt{hideothersubsections} -- Este parâmetro esconde as \emph{subseções} que não pertencem a seção atual da barra lateral.
      \item \texttt{hideallsubsections} -- Este parâmetro oculta \emph{todas} as subseções da barra lateral.
      \item \texttt{width} -- Define a largura da barra lateral, o padrão é:
      	    \verb"2.5\baselineskip"
      \item \texttt{height} -- Define a altura da barra superior, o padrão é:
      	    \verb"2.5\baselineskip"
      \item \texttt{left} -- Define que a barra lateral fique a esquerda (padrão).
      \item \texttt{right} -- Define que a barra lateral fique a direita.
    \end{itemize}

\end{frame}
